A web app that mimics an E-commerce store admin dashboard, using React, HTML, and Tailwind CSS with Next.js. It lets users explore, add, edit and delete products, with features like user authentication with JWT and Cookies and data charts using Chart.js. Product data and authentication comes from an external API.
